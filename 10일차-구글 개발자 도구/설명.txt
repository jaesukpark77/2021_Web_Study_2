구글 개발자 도구

구글크롬이 설치가 되어 있어야 한다

★ 단축키 F12 or ctrl + shift + i 

단축키를 눌러도 안되는경우
크롬 오른쪽 상단... 버튼 눌러준 다음
도구 더보기에 마우스 올리고 개발자 도구 눌러주면 ok

개발자 도구 실행해준 다음 다시 사이트 오른쪽 상단 톱니바퀴 모양 옆... 버튼 클릭 후 -DOCK SIDE 부분에서 제일 왼쪽 창이 
2개가 겹쳐져 있는 버튼 눌러주기(개발자 도구 웹사이트와 분리시켜주는 기능)

개발자 도구 폰트크기 조절하기 
CTRL + +
CTRL + -

--------------------------------------------------

★ 우클릭 막힘 풀기
원하는 사이트에서 개발자 도구 실행해준 다음
- 오른쪽 상단 톱니바퀴 모양 클릭
- 마우스 스크롤을 쭉 내려서
- Debugger 부분으로 와서 disable javascript 체크

웹사이트 마우스 우클릭이 막혀있는 사이트, 글자 클릭 드래그사 안되던 부분 해결 가능
-----------------------------------------------------------
favicon
html파일 title 밑에 <link href="사진파일명" rel="icon">

favicon은 보통 <head> 부분에 작성한다
내가 favicon을 다운로드 하길 원하는 사이트가 있다면
개발자 도구를 열어준 다음 head 부분을 확인해서
<link rel="icon">이라는 부분이 나올때 내용을 전부 지워준 다음
link부분을 나왓으면 href 부분으로 가서
마우스 우클릭 open in new tab 눌러서 다른 이름으로 저장

----------------------------------------------------------
★원하는 사이트 접속후 개발자 도구 열어준 다음

window : ctrl + shift + p
mac : 개발자 도구 상단 메뉴들 중 Sources 버튼 눌러준 다음 - Run command 부분 단축키 눌러주기

검색창 뜨면 capture full size screenshot 입력
- 내가 원하는 웹사이트에 전체 사진을 갭쳐해주는 기능
(모든 웹사이트가 깔끔하게 찍히는건 아닙니다!)
----------------------------------------------------------
윈도우 : 원하는 웹사이트 가서 ctrl+s 눌러주기
맥 : cmd + s
(모든 웹사이트에서 다 잘되는건 아니다)
html 파일을 받아주면 폴더도 같이 생기는데 이 폴더 안에는 그 웹사이트에서 사용한 사진의 90%가량이 들어있다

★★ 개발자 도구 사용시 브라켓 미리보기 기능으로 html 파일 열지말고 그냥 파일 더블클릭해서 열어주는거 추천!!!
----------------------------------------------------------
개발자 도구 실행후 내가 원하는 html태그를 선택해준 다음
오른쪽 css부분에서 px값이 입력된 부분을
클릭해서 값을 수정해주면 웹사이트에서 수정된 부분을 바로바로 확인할 수 있따
★ 주의할점
이 수정은 새로고침을 해주면 다시 원상태로 돌아오기 때문에 일시적인 것이다

개발자 도구에서 css값 수정시 키보드에 있는 방향키를 위아래 한번씩
눌르거나 꾹 눌러주면 1px 단위로 미세하게 값을 수정해주는게 가능하다

→

shift키를 누른 상태에서 방향키를 눌러줄 시 = 10px 단위로 값 변경
ctrl키를 누른 상태에서 방향키를 눌러줄 시 = 100px 단위로 값 변경

css 비활성화를 원할 시 css 속성 왼쪽 체크박스 해제 해주면 ok!

css 속성 추가를 원할 시 css 속성 작성된 부분 빈 공간 아무곳이나 클릭해주면 새로운 css속성 작성할 수 있는 공간 생성됨

개발자 도구에서 html 태그 추가, 제거, 수정이 하고 싶다면
원하는 html 태그 선택후 마우스 우클릭 edit as html 클릭!
(웬만하면 body에 마우스 우클릭 추천!)

★★ 개발자도구 오른쪽 하단 도형 그림 =
내가 선택한 태그의 가로세로값, padding값, border값, margin값을
그림으로 보여주는 도형(대상 태그에 정확히 어떠방향에 무슨값이 얼만큼 들어가 있는지 직관적으로 확인이 가능하다!)

indent guides 플러그인
내가 작성한 html태그의 시작과 끝지점 및 기타 가이드 선을 제공해주는 플러그인

설치후 왼쪽 상단 보기 버튼 클릭해서 제일밑에 indent guides번튼 눌러서 활성화 시켜준다.
