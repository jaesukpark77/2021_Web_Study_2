원하는 폰트 적용방법

구글폰트 사이트 접속후 원하는 폰트 선택 - 폰트페이지로 이동하면 폰트 옆 select this style 전부 눌러주기
- 오른쪽에 사이드 메뉴가 뜨면 하단부에 link라고 적혀져 있는 구문 전부 복사 - 자신의 html 파일 title 밑에 붙여넣기
- 다시 사이트로 가서 복사해줬던 link 부분 밑에 font-family 부분 복사 - html css로 와서 적용 원하는 태그에 붙여넣기!

적용한 폰트 굵기 바꿔주는 방법
font-weight:값(px붙이지 말 것)

★ 원하는 폰트 스타일 눌렀는데 오른쪽 메뉴바가 안나온다면? 1시방향 사각형 3개 + 모양 1개 버튼을 눌러주면 사이드 메뉴 나옵니다

★★ 폰트 사용시 태그 자체에 폰트에 관한 고유의 특징이 있는 폰트는 구글폰트 적용시 글자가 깨지는 현상이 있다.
그래서 이런경우에는 폰트 스타일이 하나 밖에 없다고 해도 font-weight 사용해줄 것!
ex) h태그

★★ 웹사이트 작업시 괜찮은 폰트

한국어 - Noto Sans KR / Nanum 시리즈

영어 - Roboto / Lato / Noto Sans
