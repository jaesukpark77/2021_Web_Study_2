지도사용법

준비물
1. 닷홈

2.파일질라

3. 자신의 도메인 파일질라에서 접속

4. 네이버 검색창에 "네이버 클라우드 플랫폼"검색후 사이트 접속

5. 회원가입 진행(네이버 아이디 있으면 연동해주는 개념) 후 로그인까지

6. 로그인 진행하신 다음 로그아웃과 랭귀지 버튼 사이에 [결제수단등록] 버튼이 있으면(없으면 그냥 넘어가면된다) 버튼 눌러서 등록해주면 된다.
  돈 나가는건 아니고 등록만 진행하면 된다(등록안하면 진행이 x)

7. 등록진행후 오른쪽 상단 Console 버튼 눌러주기

8. 새로운 창이 하나가 뜨면 왼쪽 사이드 메뉴 중 Products&Services 버튼 눌러주기

9. 오른쪽에 메뉴목록이 뜨면 AI NAVER API 버튼 눌러주기

10. 하단에 파란색 Application 등록 버튼 눌러주기

11. application 이름 부분에 영어로 이름 작성
    중간에 Maps 부분에 Web Dynamic Map 체크해주기

12. 사이트 하단부 Web 서비스 URL 부분에 자신의 도메인 주소 입력해주고 입력할 때 
    http:// 앞에다 입력해줄 것. 입력이 완료되면 추가 누르고 등록 눌러주기

13. 등록이 완료가 되면 자신이 등록해준 app이 확인되는데 서비스 구분에 책모양 아이콘 눌러주기

14. 책모양 아이콘 클릭후 사이트가 뜨면 바로가기 부분에 파란색 Web Dynamic Map API 버튼 눌러주기

15. 네이버맵 사이트가 뜨면 사이트 중앙에 시작하기 버튼 눌러주기

16. index.html이라는 이름에 html 파일 생성후 아무것도 적지 말것!

17. hello world 라고 적혀져 있는 부분 밑에 html 내용 전부 복사해서 html파일에 붙여넣기

18. 14번에 책모양 아이콘 클릭할 때의 사이트로 와서 인정정보(열쇠모양) 아이콘 클릭하기

19. 팝업창이 하나 뜨면 client id 부분 오른쪽 문서 모양 아이콘 클릭해주기

20. 복사가 완료되었으면 자신의 html 파일로 와서 title 밑에 script 부분 끝에 YOUT CLIENT ID 부분 지워주신 다음 붙여넣기

21. 붙여넣기가 완료되었으면 HTML 파일 자신의 파일질라 서버 html 폴더에 업로드해주기

★ 지도 속성은 background 속성과 비슷하기 때문에 css속성에서 
    지도가 들어가 있는 #map한테 css속성 원하는 걸 넣어주면 언제든 수정해 줄 수 있다.
