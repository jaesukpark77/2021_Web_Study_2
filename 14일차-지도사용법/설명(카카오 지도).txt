카카오 지도

1. 닷홈 로그인

2. 파일질라 실행해서 자신의 도메인 주소 접속

3. 구글에 "카카오 지도 api" 검색후 사이트 접속

4. 사이트 접속후 버튼이 보이는데 Web 이라는 버튼 클릭

5. 사이트가 이동되면 시작하기 버튼 눌러주기

6. 사이트가 이동되면 카카오 개발자사이트(파란색) 버튼 클릭

7. 새로운 사이트가 뜨는데 그 사이트에서 로그인 버튼 눌러서 로그인 해주기(자신의 카카오톡 아이디로)

8. 로그인 이후 메인페이지로 이동되었다면 오른쪽 상단 메뉴들 중에 "내 애플리케이션" 버튼 눌러주기

9. 애플리케이션 추가하기 버튼 눌러주기

10. 팝업창이 뜨면 앱 이름 사업자 명 원하는 이름 넣어주고 저장

11. 새로 추가해준 애플리케이션 클릭

12. 새로운 페이지로 이동하면 중앙에 플랫폼 부분 - 플랫폼 설정하기 버튼 눌러주기

13. web 플랫폼 등록 버튼 눌러주기

14. 팝업창이 뜨면 자신의 도메인주소 넣어주기 https:// 반드시 넣어주기

15. 등록이 모두 완료되었다면 왼쪽 사이트 메뉴 중 앱 키 버튼 눌러주기

16. 6번 과정 진행할 때 사이트 (카카오 지도 api 사이트)로 와서 중간에 !DOCTYPE html 부분 코드 전부 복사해서 자신의 html 파일에
    붙여넣기

17. 15번 사이트에서 JavaScript 부분 키 복사해서 자신의 html 파일 <div id="map"> 밑에 script 부분에 발급받은 APP KEY를 넣으시면 됩니다.
     위 구문이 적혀져 있는 부분까지 전부 지워준 다음 복사한 키 붙여넣기

18. 완료되면 파일질라 서버에 업로드해서 자신의 도메인 주소로 들어가서 지도가 제댜로 나오는지 확인!

혹시 원하는 장소를 지도에 띄우고 싶으면 어떻게 하나요???

- 우클릭시 제일 위에 숫자가 적혀있는 메뉴바가 마우스 커서 옆에 뜨는데 제일 위에 숫자값(위도 경도) 복사해서 자신의 html 파일 위도 경도 
적혀져 있는 부분에 붙여넣기 해주면 지도에서 원하는 지점 선택가능
