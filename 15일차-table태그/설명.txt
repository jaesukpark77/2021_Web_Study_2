table 태그
표를 만들어주는 태그

table:
표를 작성하기전 기본적으로 작성해주는 태그
(ul, main, form 태그들처럼 그냥 작성해주는 역할)

tr:
행을 열어주는 태그
(border 속성을 넣어줘도 적용되지 않는다)

td:
행 안에서 열 역할을 하는 태그

(그냥 사용하면 겉으로 보기에 단순한 글자와 차이가 없기 때문에
td한테 border 속성을 꼭 넣어서 형태를 잡아주는걸 권장)

★table 태그의 작성한 태그들은 단 하나의 태그 종류한테 width값을 넣어줘도 다른 태그들도 동시에 적용된다.
  (예외도 존재한다 / height 값은 적용 x)

th:
표의 부제목
특징: 글씨체가 굵으며 글씨가 중앙정렬이 된다.

colspan
:가로 열을 병합해주는 기능으로 입력한 값 만큼 병합이 이루어진다
<td colspan="값">

★ 값을 적을때 해당 table 태그에 작성한 td의 최대 갯수만큼만 적을 수 있다. 값을 초과해서 적어줘도 적용 안된다.

rowspan
: 세로 열을 병합해주는 기능
★ 값을 적어줄 때 자기자신도 값에 포함해서 계산해야한다.
병합해주고 싶은 칸의 갯수보다 +1 만큼 작성해주면 된다

tip) 전체 수정 필요한 경우
     -> ctrl + h

thead
: 표 안에서 header 부분처럼 상단 역할을 담당하는 태그
표의 제목, 서브제목 등이 들어간다

tbody
표에서 본문부분을 담당하는 태그
표의 본문내용이 들어간다

tfoot
표에서 하단부를 담당하는 태그
시맨틱태그의 footer와 비슷하다


empty-cells:hide
테이블 태그의 경우 td의 내용이 비어있어도 표가 온전히
작성은 된다. 근데 해당 css속성을 사용해주면 내용이 비어있는 표가 공간은 보이지 않게 처리가 된다

border-collapse:collapse;
테이블 태그의 경우 td하나당 테두리가 하나씩 생기는 특징이 있는데 해당 css속성을 작성해주면 테두리가 단 하나만 생겨서
이중선이 아닌 선 하나로 표가 표시가 된다.
